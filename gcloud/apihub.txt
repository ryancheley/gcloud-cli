NAME
    gcloud apihub - manage Apihub resources

SYNOPSIS
    gcloud apihub GROUP [GCLOUD_WIDE_FLAG ...]

DESCRIPTION
    Manage Apihub resources.

GCLOUD WIDE FLAGS
    These flags are available to all commands: --help.

    Run $ gcloud help for details.

GROUPS
    GROUP is one of the following:

     addons
        Manage Addon resources.

     api-hub-instances
        Manage Api Hub Instance resources.

     apis
        Manage Api resources.

     attributes
        Manage Attribute resources.

     curations
        Manage Curation resources.

     dependencies
        Manage Dependency resources.

     deployments
        Manage Deployment resources.

     discovered-api-observations
        Manage Discovered Api Observation resources.

     external-apis
        Manage External Api resources.

     host-project-registrations
        Manage Host Project Registration resources.

     operations
        Manage Operation resources.

     plugins
        Manage Plugin resources.

     runtime-project-attachments
        Manage Runtime Project Attachment resources.

NOTES
    This variant is also available:

        $ gcloud alpha apihub
