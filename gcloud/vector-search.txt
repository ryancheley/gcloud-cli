NAME
    gcloud vector-search - manage Vector Search data objects

SYNOPSIS
    gcloud vector-search GROUP [GCLOUD_WIDE_FLAG ...]

DESCRIPTION
    Manage Vector Search data objects.

GCLOUD WIDE FLAGS
    These flags are available to all commands: --help.

    Run $ gcloud help for details.

GROUPS
    GROUP is one of the following:

     collections
        Manage Collection resources.

     operations
        Manage Operation resources.

NOTES
    This variant is also available:

        $ gcloud beta vector-search
